## 🎯 Основные Возможности

### 1. ✅ Генерация Сигналов (4 Сетапа)

**Реализованные стратегии:**

1. **Breakout** - прорыв волатильности с объёмом
   - Bollinger Bands прорыв
   - Подтверждение объёмом
   - Направленность свечи

2. **Pullback** - откат в тренде
   - EMA50/200 определение тренда
   - RSI oversold/overbought
   - Вход на коррекции

3. **Mean Reversion** - возврат к средней
   - Экстремальные отклонения от BB
   - RSI в крайних зонах
   - Контртрендовая торговля

4. **Volatility Expansion** - BB Squeeze
   - Сжатие волатильности
   - Прорыв с объёмом
   - Начало движения

**Confidence Score:**
- Система вычисляет уверенность как % согласных сетапов
- 0.25 = 1/4 сетапов ✅
- 0.50 = 2/4 сетапов ✅✅
- 0.75 = 3/4 сетапов ✅✅✅
- 1.00 = 4/4 сетапов ✅✅✅✅

### 2. ✅ Бэктест Движок

**Возможности:**

- ✅ Векторный бэктест (быстро на pandas)
- ✅ Симуляция реальных условий:
  - Комиссии (0.1%)
  - Slippage (0.05%)
  - ATR-based TP/SL
- ✅ Управление портфелем:
  - Множественные позиции
  - Лимиты риска
  - Динамический sizing
- ✅ Детальные метрики:
  - Win Rate, Profit Factor
  - Max Drawdown, Sharpe Ratio
  - Exit type analysis (TP/SL/TIME)
  - По-монетная статистика

**Поддерживаемые режимы:**

- **Single Asset** - бэктест на одной монете
- **Multi Asset** - портфельная торговля

### 3. ✅ Мультивалютный Сканер

**Функции:**

- ✅ Сканирование 20-100 монет одновременно
- ✅ Загрузка данных с Binance через ccxt
- ✅ Генерация сигналов для всех монет
- ✅ Ранжирование по качеству
- ✅ Экспорт результатов в JSON
- ✅ Telegram уведомления
- ✅ Непрерывный режим (continuous scan)

**Режимы работы:**

1. **scan** - одноразовое сканирование
2. **continuous** - непрерывное с интервалом
3. **backtest** - тестирование на истории

### 4. ✅ Ранжирование Сигналов

**Критерии оценки:**

- **Confidence** (40%) - уверенность системы
- **Risk/Reward** (30%) - соотношение TP/SL
- **Active Setups** (20%) - количество сетапов
- **Asset Quality** (10%) - ликвидность, волатильность

**Функции:**

- Комплексная оценка 0-1
- Сортировка по качеству
- Объяснение оценки (debugging)

### 5. ✅ Конфигурация

**3 готовых пресета:**

| Пресет | Min Confidence | BB Std | Volume Threshold | Подходит для |
|--------|---------------|--------|------------------|--------------|
| **Conservative** | 0.75 (3/4) | 2.5 | 2.0x | Новички, низкий риск |
| **Balanced** | 0.50 (2/4) | 2.0 | 1.5x | Большинство случаев |
| **Aggressive** | 0.25 (1/4) | 1.5 | 1.2x | Опытные, высокий риск |

**Гибкая настройка:**

- Все параметры через CLI arguments
- Кастомные конфиги в коде
- Отключение отдельных сетапов

## 📊 Ответы на Ваши Вопросы

### ❓ Можно ли протестировать на истории?

**✅ ДА!** Система работает на исторических данных:

```powershell
# Бэктест на 180 днях
python run_statistical_backtest.py --symbol BTCUSDT --days 180
```

**Как это работает:**
1. Загружаются исторические свечи
2. Система идёт свеча за свечой
3. На каждой проверяет условия сигналов
4. Симулирует открытие/закрытие позиций
5. Вычисляет метрики (Win Rate, PF, DD)

### ❓ Как работать с 20-100 монетами?

**✅ РЕАЛИЗОВАНО!** Мультивалютный подход:

```powershell
# Скан 20 монет одновременно
python run_statistical_scanner.py --symbols [список] --max-signals 10
```

**Преимущества:**
- ✅ Диверсификация рисков
- ✅ Больше возможностей для входа
- ✅ Ранжирование по качеству
- ✅ Выбор лучших сигналов

**Портфельный бэктест:**

```powershell
# Тест портфеля на 10 монетах
python run_statistical_backtest.py --mode multi --symbols [список]
```

Система симулирует:
- Одновременные позиции на разных монетах
- Ротацию капитала
- Лимиты на количество позиций

## 🎓 Как Использовать

### Базовый Workflow:

```
1️⃣ Бэктест
   ↓
   Проверяем систему на истории (180 дней)
   Анализируем метрики

2️⃣ Оптимизация (если нужно)
   ↓
   Меняем preset / параметры
   Повторяем бэктест

3️⃣ Live Scanning
   ↓
   Запускаем непрерывное сканирование
   Получаем сигналы в реальном времени

4️⃣ Интеграция с XGBoost (будущее)
   ↓
   Добавляем фильтр ML моделями
```

### Примеры Команд:

**Быстрый тест системы:**
```powershell
python run_statistical_scanner.py --symbols BTCUSDT ETHUSDT SOLUSDT --max-signals 5
```

**Полный бэктест:**
```powershell
python run_statistical_backtest.py --mode multi --days 180 --symbols BTCUSDT ETHUSDT SOLUSDT NEARUSDT ADAUSDT
```

**Production режим:**
```powershell
python run_statistical_scanner.py --mode continuous --telegram --interval 1800
```
